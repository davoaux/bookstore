### List all orders

- Method: GET
- URL: localhost:8080/orders/


### Create order

- Method: POST
- URL: localhost:8080/orders/
- Body (example):

```
[
    {
        "book": "58716995-b335-4bb0-89c1-3503bc003118",
        "quantity": 1
    },
    {
        "book": "e415e3af-e87e-47e6-9bf2-f08c72e2f281",
        "quantity": 3
    }
]
```
